# Roadmap

## v0.1.0 — MVP (attuale / current)

- [x] Portale eventi con design system .italia
- [x] Registrazione partecipanti GDPR-compliant
- [x] Integrazione Jitsi Meet via IFrame API
- [x] Q&A con sistema di upvote
- [x] Controlli moderatore (mute, chat, recording toggle)
- [x] Email conferma con allegato iCal
- [x] Email reminder pre-evento
- [x] Pulizia automatica dati (GDPR retention)
- [x] Pannello admin con autenticazione
- [x] Interfaccia bilingue IT/EN
- [x] Container Docker production-ready (~200 MB)
- [x] Pipeline CI con lint, test, security scan

## v0.2.0 — In sviluppo / In development

- [ ] Sala d'attesa pre-evento con countdown
- [ ] Helm chart production-grade
- [ ] Metriche Prometheus e dashboard Grafana
- [ ] Upload immagini evento (Azure Blob Storage)
- [ ] SPID/CIE autenticazione moderatori (opzionale)
- [ ] Notifiche in-app per moderatori

## v0.3.0 — Pianificato / Planned

- [ ] Live streaming per audience > 300 (HLS/DASH)
- [ ] Breakout rooms
- [ ] Sondaggi e polling integrati
- [ ] Trascrizione automatica (Whisper)
- [ ] Export report evento (PDF)
- [ ] API pubblica per integrazione con altri portali PA
